В приведенной  последовательности, по-видимому, чередуются два значения: 0 и дробью вида 2/n, где n - возрастающее нечетное целое число. Давайте разберем эту схему:

1-й член: 2/1 = 2
2-й член: 0
3-й член: 2/3
4-й член: 0
5-й  член: 2/5
6-й  член: 0
7-й  член: 2/7
...

Похоже, что все четные члены равны 0, а нечетные имеют вид 2/n.



Можно использовать функцию floor, которая непосредственно дает n-й член без учета того, является ли n четным или нечетным

a(n) = 2 / (2 * floor(n / 2) + 1)

 Функция "floor", по сути, округляет до ближайшего целого числа. При нечетном значении 'n' знаменатель равен '2 * n + 1', что дает дробь вида 2/n. При четном значении 'n' знаменатель становится равным '2 * n + 1', что упрощается до 0.
